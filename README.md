# Catch The Jerry
## 2D Game With Kotlin

This is my first application while learning Kotlin. This is a 2D game. 

I learned how to make a logo and add it to the application.

I used 
- AlertDialog, 
- Toast, 
- CountDownTimer, 
- Runnable 
structures while making this game.
